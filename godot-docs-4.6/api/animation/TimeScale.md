# AnimationNodeTimeScale

## Meta

- Name: AnimationNodeTimeScale
- Source: AnimationNodeTimeScale.xml
- Inherits: AnimationNode
- Inheritance Chain: AnimationNodeTimeScale -> AnimationNode -> Resource -> RefCounted -> Object

## Brief Description

A time-scaling animation node used in AnimationTree.

## Description

Allows to scale the speed of the animation (or reverse it) in any child AnimationNodes. Setting it to 0.0 will pause the animation.
