## Brief Description

Texture for Cubemap that is bound to a texture created on the RenderingDevice.

## Description

This texture class allows you to use a cubemap texture created directly on the RenderingDevice as a texture for materials, meshes, etc. **Note:** TextureCubemapRD is intended for low-level usage with RenderingDevice. For most use cases, use Cubemap instead.
